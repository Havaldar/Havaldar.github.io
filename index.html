clear: This will clear the screen for you.

ls: This will give you a list of all the directories and files in the working directory.

cd: directory_name: This will change directory such that you will be in your specified directory.

cd ..: Use this command to go back one directory.

cd: By typing just cd you will go to the root directory.

pwd: This command will tell you the current directory you are in.

cat text_file: cat will print out the contents of the specified text file. Please just use it with files with a .txt extention. You can also print many files at a time simply by seperating them with spaces.

open file_name: open will open the specified file in your program of preference (i.e. this website). Use this with .file extentions. This is limited to one file at a time.
